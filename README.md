# Bitmex_HFT

The goal of this project is to explore the market microstructure of BTC traded on Bitmex. 

1. Data is collected and stored in a Arctic Tick Database by running main.py.

2. Dependencies 

Pip install:
```
Pandas
Arctic
Websocket
Json
```
